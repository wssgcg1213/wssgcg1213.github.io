重邮Mac OS X, 寝室内外网联网教程


先来看看叠加之后的效果.172网段的是内网,重邮的内网是100Mbps不限速(入楼应该是千兆,但是入户只有百兆),可以通过内网上一些比如教务在线,红岩网校的系列网站,PT啦,速度还是不错的

###You Need:

  1. a Mac
  2. 你的统一认证码和密码(用于内网)
  3. 一个用于NetKeeper上网的账号(用于外网连接)
  4. 如果你是Air或者RMBP没有RJ45网线接口,你还需要去买一个usb/雷电 转 网线接口的转换器一只
  5. 如果你的室友也有上网需求,就需要交换机一台以及连接交换机的网线若干

内网: http://pan.baidu.com/share/link?uk=3154435625&shareid=2374946882

外网: http://pan.baidu.com/share/link?uk=3154435625&shareid=2417216665

下载上面的内网文件,扩展名是mobileconfig,这种文件原先是从iPhone同步助手导出给iPhone的运营商设置文件,但是你要知道iOS是基于Mac OS X的,所以经过修改便可以导入OS X使用.

重邮使用的内网网关认证是基于MD5方式的802.1x认证,通过修改mobileconfig的字段既可破解支持.(听不懂可以忽略哈)

我们用一个文本编辑器打开这个文件,修改其中标注的统一认证码和密码,保存打开,会出现一个设置的导入界面,一路下一步并鉴权既可.

以后只要在网络设置中就可以看到802.1x的选项,在插网线的情况下可以认证了.

###关于外网

电信已经提供了Mac版本的netkeeper,在cq电信官网校园专区就可以找到,我这里另存了一份到百度网盘需要的自取

PS:这个netkeeper for Mac程序也可以在WLAN-CQUPT环境下使用.
PPS:这个程序没加密,反汇编一下就全出来了,想破解NetKeeper的可以看看

VPS的简单配置

前言

原来US的两个虚拟主机连着抽风- -, 实在受不了,最近locvps搞活动弄来一个55折的code,于是拖着@ricter还有几个逗比合租了个HK的节点,一个月掏两顿饭钱给VPS还是有的…..so, 总算是圆了自己要一个HK VPS的愿望.

不过这VPS的配置对新手来讲太痛苦了

###step1
装的是ubuntu server 12.04 32bit,然后ape-get各种runtime,花了有一个多小时

###step2
configure,nginx和apache2的设置花了一天的时候弄明白了,关键是在设置各个用户的访问权限
目前还在混乱中,nginx暂以root权限运行 apache2,php-fcgi以www-data权限运行
还是先理一下整体架构,nginx做前端,监听80端口,在/etc/nginx/nginx.conf中设置

location ~{ proxy-pass 127.0.0.1:xxxx; }
```

apache2在8000端口监听,配置文件在/etc/apache2/httpd.conf 和/etc/apache2/sites-enabled/*
php5的php.ini文件在/etc/php5/xxx下面,这里好几个文件夹都包含php.ini,每次改配置都是几个全改好郁闷T.T

###Now
PHP能欢快的跑起来了,除了phpmyadmin的配置有点小问题
由于是合租VPS,所以每个人都分配了管理员权限,接下来要考虑怎样让每个人的操作尽量不影响其他人

TASKINFO: http://vps.zeroling.com/taskinfo.php
